# Mora - Domain Time Tracker Installation

## Quick Setup

1. **Generate Icons** (one-time setup):
   - Open `create-icons.html` in Chrome
   - It will automatically download 3 icon files
   - Save them in the Mora folder

2. **Install Extension**:
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (top right toggle)
   - Click "Load unpacked"
   - Select the Mora folder

3. **Usage**:
   - Pin the extension to your toolbar
   - Browse normally - time tracking is automatic
   - Click the extension icon to view your domain usage

## Features

- ✅ Automatic domain time tracking
- ✅ Daily usage statistics
- ✅ Top 10 most visited domains
- ✅ Clean, minimal interface
- ✅ Local storage only (privacy-first)
- ✅ Clear data option

## How it Works

The extension tracks:
- Active tab domain changes
- Time spent on each domain
- Daily usage patterns
- Stores data locally in Chrome storage

Perfect for understanding your browsing habits!
